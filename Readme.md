## üîç **Searching for Family Members**

### **Basic Search**
1. **Enter Search Terms**: Type a name or ID in the search box
2. **View Results**: Browse through the search results
3. **Select Person**: Click on any person to view their details

### **Search Tips**
- **Partial Names**: You can search with just part of a name
- **ID Search**: Use the person's ID for exact matches
- **Case Insensitive**: Search works regardless of letter case

### **Search Results**
Each result shows:
- **Name**: Full name of the person

- **Basic Info**: Birth year, death year, gender
- **Action Buttons**: View different types of trees

## üå≤ **Tree Visualization Modes**

### **1. Basic Tree View**
**Access**: Click "View Tree" button on any person

**Features**:
- Interactive hierarchical tree layout
- Click nodes to expand/collapse branches
- Side panel with person details
- Zoom and pan controls

**Best For**: Quick family overview, simple relationships

### **2. Enhanced Tree View**
**Access**: Click "Enhanced Tree" button on any person

**Features**:
- Complex relationship mapping
- Color-coded nodes by gender and status
- Comprehensive relationship panel
- Advanced interaction controls

**Best For**: Detailed family analysis, complex relationships

### **3. Unified Tree View**
**Access**: Click "View Unified Tree" button on any person

**Features**:
- Combined features from both modes
- Professional genealogy chart layout
- Export capabilities
- Minimap for navigation

**Best For**: Comprehensive family exploration

## üìà **Understanding Family Relationships**

### **Direct Relatives**
- **Parents**: Father and mother
- **Children**: Sons and daughters
- **Spouses**: Husbands and wives
- **Siblings**: Brothers and sisters

### **Extended Family**
- **Grandparents**: Parents of parents
- **Grandchildren**: Children of children
- **Uncles/Aunts**: Parents' siblings
- **Cousins**: Children of uncles/aunts
- **Nephews/Nieces**: Children of siblings

### **Relationship Degrees**
- **1st Degree**: Parents, children, spouses
- **2nd Degree**: Grandparents, grandchildren, siblings
- **3rd Degree**: Great-grandparents, great-grandchildren, uncles/aunts
- **4th Degree**: Great-great-grandparents, cousins, nephews/nieces

## üîß **Troubleshooting Common Issues**

### **Tree Not Loading**
- **Check Internet Connection**: Ensure stable internet
- **Refresh Page**: Try reloading the page
- **Clear Browser Cache**: Clear cache and cookies
- **Try Different Browser**: Test with another browser

### **Search Not Working**
- **Check Spelling**: Verify search terms are correct
- **Try Partial Names**: Use shorter search terms
- **Check Database**: Ensure data is properly loaded
- **Contact Administrator**: If issues persist

### **Visualization Problems**
- **Update Browser**: Ensure browser is up to date
- **Enable JavaScript**: Check JavaScript is enabled
- **Check Console**: Look for error messages
- **Try Different View**: Switch between tree modes

### **Performance Issues**
- **Large Trees**: Very large families may load slowly
- **Close Other Tabs**: Free up browser memory
- **Wait for Loading**: Allow time for complex trees to render
- **Use Search**: Find specific people instead of loading entire tree
